# BYTZ

BYTZ is a modern, fast, and secure URL shortener built with a minimal interface, link vault management, and fast link generation. 

## Features

- **Link Generation**: Forge short, powerful BYTZ URLs instantly in a clean, minimalist space.
- **My Vault**: Keep track of and manage all your shortened URLs locally.
- **Smart Selection**: Select individual links or all links to quickly delete and manage your history.
- **Auto-Closing SaaS Modals**: Beautiful, blurred-backdrop modals to instantly preview and copy your new links.
- **Undo Actions**: Accidentally deleted a link? Instantly undo deletions with a 7-second grace period.

## Getting Started

First, run the development server:

```bash

# or
yarn dev
# or

# or
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
