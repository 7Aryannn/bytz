# BYTZ

🔗 **Live Link:** [https://bytz-url.vercel.app/](https://bytz-url.vercel.app/)

BYTZ is a modern, fast, and secure URL shortener built with a beautifully crafted minimal interface, advanced link vault management, detailed analytics, and fast link generation.

## ✨ Features

### 🚀 Link Generation
- **Instant Forging:** Create short, powerful BYTZ URLs instantly in a clean, minimalist space.
- **Custom Aliases:** Personalize your links with optional custom aliases (e.g., `bytz.io/my-custom-name`).
- **Auto-Closing SaaS Modals:** Beautiful, blurred-backdrop modals feature an animated 10-second border countdown to instantly preview and copy your new links.
- **One-Click Copy & Toast Notifications:** Seamlessly copy your new links to the clipboard, accompanied by sleek toast notifications with animated timer bars.

### 🗄️ My Vault
- **Local Link History:** Keep track of and meticulously manage all your shortened URLs locally.
- **Smart Selection & Bulk Actions:** Select individual links, or use keyboard shortcuts (`Ctrl + A`) to select all links to quickly delete and manage your history.
- **Undo Actions:** Accidentally deleted a link? Instantly restore it using the undo toast notification offering a 7-second grace period.

### 📊 Dashboard & Analytics
- **Usage Tracking:** Monitor your link generation usage against your BYTZ limit with a visual progress bar.
- **Detailed Analytics Overview:** View comprehensive dummy statistics for your shortened links:
  - **Total Clicks:** Track engagement with your links.
  - **Top Countries:** View geographic distribution of your link traffic with visual percentage bars.
  - **Device Types:** Understand if your audience uses Mobile, Desktop, or Tablet devices.
- **Link Management:** Easily update the destination URL of an existing shortened link, or safely delete links with secure confirmation modals to prevent accidental removal.

## 💻 Getting Started

First, clone the repository and navigate into the project directory:

```bash
git clone <your-repo-url>
cd bytz
```

Then, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser to see the result.
